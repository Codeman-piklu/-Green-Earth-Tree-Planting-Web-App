
# 🌱 Green Earth – Tree Planting Web App
![Banner](./assets/Screenshot%202026-03-26%20235516.png)

A modern, interactive **tree gallery and planting web application** built with **HTML, TailwindCSS, DaisyUI, and JavaScript**. Explore trees, manage your cart & wishlist, apply discounts, and see detailed plant info—all with smooth animations and responsive design.

This project demonstrates **front-end development skills**, including API integration, dynamic DOM manipulation, responsive design, and interactive UI components.

---

## 🚀 Features

### 🌿 Gallery & Categories

* Dynamic tree gallery fetched from a **public API**.
* Filter trees by **category** or **search by name**.
* Sort trees by **price (low → high / high → low)** and **name (A → Z)**.
* “All Plants” button to quickly reset filters.

### 🛒 Cart System

* Add/remove trees to/from cart.
* Increase/decrease quantity dynamically.
* **Progress bar** to unlock discounts based on total amount.
* Tiered discounts:

  * 10% ≥ 3,000৳
  * 20% ≥ 5,000৳
  * 25% ≥ 8,000৳
  * 30% ≥ 10,000৳
  * 40% ≥ 12,000৳
* Coupon codes: `SAVE10` (10%), `SAVE20` (20%).

### ❤️ Wishlist

* Add/remove plants from wishlist.
* Wishlist count displayed in navbar.
* Modal displays wishlist items with remove option.
* Wishlist stored in **localStorage** for persistence.

### 🖼 Plant Details Modal

* Click **Details** for enlarged image, full description, category, and price.
* Reusable modal for all plants.

### 🎨 UI & Responsiveness

* Clean, modern layout using **TailwindCSS + DaisyUI**.
* Mobile-friendly and responsive grid layouts.
* Smooth hover effects and interactive buttons.
* Animated banner highlighting offers.

### ⚡ Notifications

* Toastify.js used for **success/error feedback**:

  * Added to cart
  * Wishlist changes
  * Coupon applied/invalid

## 🖥 How to Run Locally

1. Clone the repository:

```bash
git clone <https://github.com/Codeman-piklu/-Green-Earth-Tree-Planting-Web-App.git>
```

2. Navigate into the project:

```bash
cd green-earth
```

3. Open `index.html` in your browser.

> ✅ No backend required; all data is fetched from **OpenAPI**.
